# fab-scripts-
# Script de Ejemplo en Python

## Descripción
Este script es un programa sencillo e interactivo que sirve como introducción a Python. Solicita al usuario su nombre y su edad, procesa los datos y muestra un saludo personalizado indicando si la persona es mayor o menor de edad.

## Requerimientos
* **Python 3.x**: El script utiliza únicamente funciones estándar integradas en el lenguaje, por lo que no requiere la instalación de librerías externas.
